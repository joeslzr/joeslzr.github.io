Analog Light Stick


Abstract

A button is simply defined as "a knob on a piece of electrical or electronic equipment that is pressed to operate it.". By this definition, we are afforded as fair bit of leeway when choosing an input method for our lighting apparatus - so long as the input can be justified as "pressing to operate". If we were to further explore the term "press" we are afforded even more options. Which leads to the next assignment specification presented: is one LED enough? Clearly with so many input options the answer is no. In fact, the average Canadian household has 26.4 light bulbs so why are we being herd like sheep to the single press button industry? I say nay! Thus, I present an alternative to the humble light switch: the light joystick. With this analog based input, not only are we able to control multiple lights at once, we can seamlessly alternate between them. That being said, there is still a standard button included to appease the single input purists.
